# ParkPal — Peer-to-Peer Driveway Parking Marketplace

**"Your Driveway. Their Parking."**

A modern, production-ready peer-to-peer parking platform that connects homeowners (Hosts) with drivers (Guests). Hosts monetize their unused driveways, yards, and garages while drivers find affordable parking near events, stadiums, airports, schools, and busy areas.

## Key Features

### For Guests (Drivers)
- Interactive dark-mode map with real-time availability
- Smart search & filters (price, distance, location)
- **Instant 2-tap Stripe checkout** — book in under 60 seconds
- Get directions (Google/Apple Maps)
- Booking history
- **PWA Support** — "Add to Home Screen" for native-like mobile experience

### For Hosts (Homeowners)
- Easy spot listing with map location picker
- Flexible hourly & event pricing
- Instant availability toggle + auto-off timer
- Earnings dashboard with monthly charts
- Promote spots for better visibility ($5–$20 packages)
- Violation reporting

### Platform
- Secure JWT authentication (role-based)
- **85/15 revenue split** (Hosts keep 85%)
- Fully responsive design
- Terms of Service & Privacy Policy included

---

## Tech Stack
- **Frontend:** React 19, Tailwind CSS, Shadcn/UI, Leaflet, Recharts
- **Backend:** FastAPI (Python), MongoDB
- **Payments:** Stripe Checkout (fully working)

---

## Revenue Model
- 15% platform fee on bookings (Hosts keep **85%**)
- Promoted listings: $5 (24h), $12 (3 days), $20 (7 days)
- Zero listing fees
